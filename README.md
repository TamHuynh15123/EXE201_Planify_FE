# Planify FE

Frontend for Planify project.

## Folder Structure (Clean Architecture)

Based on the provided design:

- `src/assets`: Images, fonts, and other static assets.
- `src/components`: Reusable UI components.
- `src/layout`: Layout components (Header, Footer, etc.).
- `src/pages`: Application pages and routes.
- `src/features`: Feature-based modules.
- `src/hooks`: Custom React hooks.
- `src/context`: React context for global state.
- `src/redux`: Redux store, slices, and logic.
- `src/services`: API calls and external services.
- `src/utils`: Helper functions and utilities.

## Getting Started

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
